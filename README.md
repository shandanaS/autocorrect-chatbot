# 🧠 Text Autocorrect Chatbot

A word-level autocorrect chatbot built in Google Colab using Norvig's probabilistic spell corrector.

## Features
- Loads a real English corpus (big.txt)
- Corrects spelling using edit distance
- Interactive chatbot interface

## How to Run
1. Open the `.ipynb` file in Google Colab
2. Run all cells
3. Type in the chatbot and get autocorrected output

(https://colab.research.google.com/github/shandanaS/autocorrect-chatbot/blob/main/autocorrect_chatbot.ipynb)
